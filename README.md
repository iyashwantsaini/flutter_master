# Flutter Commands

### Install
* Flutter
* Android Studio
* Android Studio Dart & Flutter Plugin
* VsCode Dart & Flutter Extension 
* Java for Gradle

### Init
* which flutter
* flutter doctor
* flutter devices

### New Project
* flutter create app_name
* cd app_name
* flutter run
* flutter run -v | For seeing background tasks

### Gradle Commands
* ./gradlew --stop | To stop all processes
* ./gradlew --status | To view running
